# 🍽 Neemble Eat - Frontend

Neemble Eat is a **QR code-based ordering system for restaurants in Luanda, Angola**. Customers scan a QR code at their table to access the menu, place orders, and track their order status in real-time.

🚧 **This repository contains the frontend of Neemble Eat and is currently in development.** 🚧

## 🌐 Official URL
🔗 [Neemble Eat](https://www.neemble-eat.ao/home)

---

## 🚀 Tech Stack
- **Framework:** React (TypeScript)
- **Styling:** Tailwind CSS
- **State Management:** React Context API
- **Routing:** React Router
- **API Communication:** Fetch / Axios

---

## 🔧 Installation & Setup

### 🛠 **Local Development**
1. Clone the repository:
   ```sh
   git clone https://github.com/your-repo/neemble-eat.git
   cd neemble-eat
   ```
2. Install dependencies:
   ```sh
   npm install
   ```
3. Create a `.env` file based on `.env.example` and configure necessary environment variables.

4. Start the development server:
   ```sh
   npm run dev
   ```
